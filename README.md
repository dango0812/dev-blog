```
📂 DEV-BLOG
┣ 📂 app/
┃ ┣ 📂 (posts)/      # 게시글 그룹 - 게시글 목록/상세 페이지
┃ ┣ 📂 admin/        # 관리자 페이지 - 대시보드/게시글 관리
┃ ┗ 📂 api/          # API Route Handler
┣ 📂 components/     # 공통 컴포넌트
┣ 📂 model/          # Zod 스키마/타입 정의
┣ 📂 utils/          # 유틸리티 함수
┣ 📂 lib/            # 외부 라이브러리 활용 함수 정의
┣ 📂 constants/      # 전역 상수
┗ 📂 hooks/          # 커스텀 훅
```
